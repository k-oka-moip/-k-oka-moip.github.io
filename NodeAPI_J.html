Node API documentation version v1.3

http://example.api.com/x-nmos/node/{version}

  • version: required(v1.3)

概要

Node APIは、そのノードがフローやソースなどのNMOSリソースを提供できるかどうかに関わらず、システム内の各NMOSノードによって公開されます。 Node APIによって公開されたデータは、Registration APIを介して（分散された）レジストリに入力するために使用されます。 小規模なデプロイメントでは、Peer to Peer仕様を実装しているノードから必要に応じて、Node APIを直接取得します。

その他のドキュメント

このAPIの動作をカバーするその他の規範的なドキュメントは、このリポジトリのdocsフォルダに含まれています。（https://amwa-tv.github.io/nmos-discovery-registration/tags/v1.3/docs/）
